# SpotifyFront

A single-file, static front-end clone of the Spotify web player interface, built with plain HTML and CSS (plus a small bit of JavaScript for filter pill interactions). It recreates the classic Spotify dark-themed layout: sidebar navigation, a home feed with recommendation cards, and a bottom player bar — purely as a visual/UI mockup with no real playback or backend.

## Features

- Spotify-style dark theme with the signature green accent color
- Sidebar navigation with icon buttons and tooltips
- Search bar and filter pills (Music / Podcasts, etc.) with click interaction
- "Made for you" style recommendation cards with cover art placeholders and hover play button
- Quick-access grid of recently played items
- Bottom player bar with track info, playback controls, progress bar, and volume/like icons
- Fully responsive layout built with CSS Grid and Flexbox
- No backend or real audio playback — this is a static UI/visual clone only

## Technologies

- HTML5
- CSS3 (CSS variables, Grid, Flexbox)
- Vanilla JavaScript (minor UI interactions)
- Google Fonts (Circular Std)

## Requirements

- Any modern web browser
- No build tools, server, or dependencies required

## Installation and Run

git clone https://github.com/RenaMehdiyeva88/SpotifyFront.git
cd SpotifyFront

Simply open `index.html` directly in your browser, or serve it with any static file server, for example:

npx serve .
